# Webbs Memorial Orphanage

A simple full-stack website for a children's home with correct English content and a small Node.js backend.

## Features

- Home page with mission, programs, donation details, and gallery
- Contact form with backend message submission API
- Admin page for viewing submitted contact messages

## Run

```bash
node server.js
```

Then open `http://localhost:3000`.

Admin page:

```bash
http://localhost:3000/admin.html
```
